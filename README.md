# teamya

## setup
```
npm install
```

### concurrently start app and json-socket-server
```
npm start
```

(this is shorthand for)

```
npm run serve 
```

and

```
json-socket-server db.json
```

### compile and minify for production
```
npm run build
```

### lint and fix files
```
npm run lint
```
